# artifacts/

This folder contains **serialised intermediate outputs** generated by the notebooks:
processed feature tables, model outputs, worker scores, and cluster assignments.

These files are the "glue" between notebooks — notebook 03 can optionally
read `worker_features.csv` produced by notebook 02 instead of recomputing it.

> **Note:** All files in this folder are **excluded from git** (see `.gitignore`).
> They can be large and are fully reproducible by running the notebooks.
> See [`docs/reproducibility.md`](../docs/reproducibility.md) for instructions.

## Schema: `worker_features.csv`

One row per worker. Index column: `ozon_id`.

| Column | Type | Description |
|---|---|---|
| `n_answers` | int | Total annotation answers |
| `n_projects` | int | Projects participated in |
| `n_pools` | int | Pools participated in |
| `lifetime_h` | float | Hours from first to last answer |
| `gold_acc` | float | Gold-task accuracy (NaN if < 5 gold tasks) |
| `n_gold` | int | Gold tasks answered |
| `reg_acc` | float | Regular-task accuracy |
| `n_regular` | int | Regular tasks answered |
| `agreement_rate` | float | Agreement with majority on regular tasks |
| `med_duration` | float | Median per-task seconds (1–600 s filter) |
| `std_duration` | float | Std dev of per-task seconds |
| `p10_duration` | float | 10th percentile per-task seconds |
| `p90_duration` | float | 90th percentile per-task seconds |
| `answer_entropy` | float | Shannon entropy of answer distribution |
| `skip_rate` | float | Fraction of tasks skipped |
| `has_regular` | int | 1 if worker appeared in any regular pool |
| `has_rehab` | int | 1 if worker was ever rehabilitated |
| `has_training` | int | 1 if worker completed training |
| `mean_price` | float | Mean task price (regular pool) |
| `max_price` | float | Max task price (regular pool) |

## How to export artifacts from notebooks

```python
# Worker features
worker_features.to_csv("artifacts/worker_features.csv", index=True)

# Worker clusters
worker_clusters.to_csv("artifacts/worker_clusters.csv", index=True)

# Composite scores
composite_scores.to_csv("artifacts/worker_scores.csv", index=True, header=["composite_score"])
```
