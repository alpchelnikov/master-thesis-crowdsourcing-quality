## How to add results

When a notebook produces a result table, export it and add it here:

```python
# In notebook — export a DataFrame
results_df.to_csv("results/baseline_metrics.csv", index=False)

# For a markdown table
with open("results/eda_key_findings.md", "w") as f:
    f.write(results_df.to_markdown(index=False))
```
